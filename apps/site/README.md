# PulseNote Site

Public landing site for PulseNote, now based on the cloned Mizu Light Astro template and adapted for the PulseNote release communication workflow.

## Stack

- Astro
- Tailwind CSS v4
- Astro Icon

## Routes

- `/`: landing page
- `/coming-soon`: hosted access placeholder
